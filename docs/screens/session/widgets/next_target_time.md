# Next Target Time Widget

`RunWidgetType.nextTargetTime` — the target clock time for completing the
current kilometer.

- **Value:** `session.nextTargetTimeDisplay`.
- **Default label:** `Next target`.
- **Style fields:** label size/color, value size/color. No status coloring.
- **Orientation:** label above value; label left / value right when placed
  1 row tall × full width.
- **Default layout:** row 2, right column, value size 24.
